Sorting

Motivation

Why would you care about sorting? To start, there may be easily accessible examples, such as in a phonebook. Sorting also makes certain computational tasks easier, such as finding the median and finding an element in a list. There are also applications in fields of computer science, such as data compression and computer graphics.

Insertion Sort

Let's begin with our first attempt at sorting.

Let's go through out array and put each element in an already sorted array.

Here's the psuedocode

For i=1...n

insert A[i] into sorted array A[0:i-1] by pairwise swaps

Insertion Sort Complexity Analysis

We can analyze the time complexity of insertion sort by determining how long the inner loop takes and how many times our inner loop executes.

The inner loop has complexity O(n), because we may have to swap the element all the way to the front of the subarray, which means O(n) comparisons and O(n) swaps. The inner loop executes O(n) times, which means insertion sort is O(n^2).
